# Stretch Goal: Blog Section 📝

Add a blog section to the existing personal branding website.

## What to Do

1. Read the existing `index.html` and `style.css`
2. Add a "Blog" link in the navigation/header of `index.html`
3. Create a new `blog.html` page with 2–3 sample blog posts
4. Style it consistently with the existing theme

## Files to Create/Modify

- **Modify** `index.html` — add a "Blog" navigation link
- **Create** `blog.html` — the blog listing page
- **Modify** `style.css` — add blog-specific styles

## Blog Page Structure (`blog.html`)

### Header
- Same header/nav as main page, with "Home" and "Blog" links
- "Blog" should be highlighted as active

### Blog Posts
Create 2–3 sample posts. Each post card should have:
- Post title (linked, clickable-looking but stays on same page)
- Date published
- A short excerpt (2–3 sentences)
- A "Read more →" link (can be a placeholder `#`)
- Optional: a category/tag label

### Sample Post Content
Use fun, relatable topics:
1. **"My Journey Into Tech"** — A short story about discovering technology
2. **"5 Things I Learned at My First Hackathon"** — Lessons from a beginner's perspective
3. **"Why Personal Branding Matters"** — Reflections on building an online presence

### Footer
- Same footer as main page

## Styling
- Match the existing theme's colors, fonts, and overall feel
- Blog post cards should be clean and readable
- Good typography: readable line-height (1.6+), comfortable max-width for text
- Responsive: single column on mobile, can be wider on desktop

## Constraints
- **HTML + CSS only.** No JavaScript for the blog.
- **Consistent with existing theme.** Reuse the same `style.css` — add blog styles at the end.
- **Navigation between pages.** Both pages should link to each other.
