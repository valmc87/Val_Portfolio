# Stretch Goal: Projects Gallery 🖼️

Add a projects/portfolio gallery section to the existing personal branding website.

## What to Do

1. Read the existing `index.html` and `style.css`
2. Add a new "Projects" section to `index.html` (before the Social Links section)
3. Style it as a visual gallery grid

## Section Structure

### Projects Gallery
A grid of 3–4 project cards. Each card has:
- A colored header/banner area (use CSS gradients — no images needed)
- Project title
- Short description (1–2 sentences)
- Tags/tech used (small pill-shaped labels)
- A "View Project →" link (placeholder `#`)

### Sample Projects
Use relatable, non-intimidating examples:
1. **"Community Recipe Book"** — A collection of family recipes shared online. Tags: HTML, CSS
2. **"Study Planner"** — A simple tool to organize study schedules. Tags: Planning, Design
3. **"Volunteer Tracker"** — Tracking volunteer hours for a local charity. Tags: Community, Data
4. **"Photo Journal"** — A visual diary of travel adventures. Tags: Photography, Creative

## Styling
- **Grid layout:** 2 columns on desktop, 1 column on mobile
- **Cards:** Match the existing theme's card style (colors, border-radius, shadows)
- **Gradient banners:** Each card's header uses a different CSS gradient as a visual placeholder
- **Hover effect:** Subtle lift or shadow change on hover
- **Tags:** Small, pill-shaped, using the theme's accent colors

## Constraints
- **Pure HTML + CSS only.** No JavaScript.
- **No external images.** Use CSS gradients for visual interest.
- **Consistent with existing theme.** Match colors, fonts, and spacing.
- **Responsive.** Grid adapts from 2 columns to 1 on mobile.
